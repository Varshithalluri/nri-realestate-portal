# NRI Real Estate Portal (Local)

Minimal local project for NRIs to list and browse properties.

## Prerequisites
- Node.js (v14+)
- MySQL server running locally

## Quick start (after step-by-step files are added)
1. `npm install`
2. Create database and tables: `mysql -u root -p < db.sql`
3. Edit DB credentials in `server.js`
4. Start server: `node server.js`
5. Open `http://localhost:3000` in your browser
